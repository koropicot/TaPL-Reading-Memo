9.2.3

```
Γ |- f x y : Bool
```
と仮定すると、その導出木(と同等の何か)は

```
Γ |- f x y : Bool by T-APP {
  Γ |- f x : T -> Bool by T-APP {
    Γ |- f : S -> T -> Bool by T-VAR {
      f: S -> T -> Bool in Γ
    }
    Γ |- x  : S by T-VAR {
      X : S in Γ
    }
  }
  Γ |- y : S by T-VAR {
    Y : T in Γ
  }
}
```
である。よって

```
Γ = f: S -> T -> Bool, x:S, y:T
```

となる。例えば

```
f x y = x == y
```
とすれば

```
Γ = f:Bool -> Bool -> Bool, x:Bool, y:Bool
```
とすることが出来る。
